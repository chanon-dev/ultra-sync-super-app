## ⚡ Tech Stack Highlights
- **Backend:** Go (Golang) Microservices, gRPC, mTLS, Apache Kafka, Vault.
- **Frontend:** Flutter, BLoC State Management, Clean Architecture, Isolates.
- **Database:** PostgreSQL (PostGIS), Redis, Redis Pub/Sub.
- **Infra:** Docker, Kubernetes (K8s), Prometheus, Grafana, Jaeger.
